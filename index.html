CopyPaste

CopyPaste

CopyPaste is a RogueLike game that was made in 24 hours, where the goal is to go the furthest possible in a dungeon.
As the name suggest, the player, when killing an enemy, copies its ability and must then plan which enemy to kill first to be able to subjugate the others more effeciently.

I had the roles of Game Designer, Game Programmer, and Producer.
